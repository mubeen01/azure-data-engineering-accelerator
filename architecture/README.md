# Architecture

The Azure Data Engineering Accelerator (ADEA) is designed using proven enterprise architecture patterns for scalable, secure, and maintainable data platforms on Microsoft Azure.

## Architecture Principles

- Modular
- Reusable
- Metadata-driven
- Secure by default
- Infrastructure as Code
- CI/CD ready
- Cloud-native
- Cost-conscious
- Production-ready

## Core Components

- Azure Data Factory
- Azure Data Lake Storage Gen2
- Azure Databricks
- Delta Lake
- Azure SQL / Microsoft Fabric
- Azure Key Vault
- Azure Monitor
- GitHub Actions
- Bicep

Refer to the remaining documents in this folder for detailed architecture guidance.
