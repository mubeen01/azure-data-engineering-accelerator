# Overview

Azure Data Engineering Accelerator (ADEA) is an open-source framework that provides reusable architecture, templates, best practices, and production-ready examples for building Azure-based data engineering solutions.

Instead of starting every project from scratch, engineers can use ADEA to rapidly deploy enterprise-grade Azure data platforms using proven patterns.

The project focuses on real-world implementation rather than isolated code samples.

# Project Structure

```text
azure-data-engineering-accelerator/

src/                  Generic reusable framework (sql/, adf/, databricks/,
                      infrastructure/, monitoring/, security/, synapse/, fabric/)
examples/             End-to-end industry implementations (banking, healthcare,
                      retail, insurance)
tools/                Dev tooling — e.g. the synthetic data generator
datasets/             Sample data (generated by tools/synthetic-data-generator)
architecture/         Design docs
docs/
assets/
scripts/
tests/
.github/
```

See [ROADMAP.md](ROADMAP.md) for what's implemented in each folder today.

# Roadmap
