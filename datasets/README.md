# Sample Datasets

This directory contains realistic sample datasets for demonstrating enterprise Azure Data Engineering solutions.

## Objectives

- Learning
- Development
- Testing
- Demonstrations
- End-to-end pipeline validation

## Available Domains

### Healthcare

- Patients
- Providers
- Claims
- Pharmacy

### Banking

- Customers
- Accounts
- Transactions
- Loans

### Retail

- Customers
- Products
- Orders
- Inventory

## Notes

All datasets are synthetic and contain no real personal or confidential information.

Files here are generated by `tools/synthetic-data-generator/` (Phase 3) rather
than committed by hand — see that tool's README for usage. As of now, only
the banking domain generator is implemented; these CSVs are still empty
until it's run.
